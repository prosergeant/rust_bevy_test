# План работ по лучшению игры "Flappy Bird" (Завершено)

Цель: Добавить полноценное меню и исправить ошибку с бесконечным созданием труб при выходе из паузы.

## [x] 1. Рефакторинг состояний игры (`GameState`)

-   **Проблема:** Текущее использование `GameState::Playing` для создания игровых объектов вызывает их повторное создание при выходе из паузы.
-   **Задача:** Разделить состояние "начала игры" и "игрового процесса".
-   **Результат:** Добавлено состояние `PreGame`, `Menu` переименовано в `MainMenu`.

## [x] 2. Обновление логики спавна объектов

-   **Проблема:** Птица и трубы создаются при входе в состояние `Playing`.
-   **Задача:** Перенести логику создания в новое состояние `PreGame`.
-   **Результат:** Птица и трубы спавнятся в `PreGame`. Добавлен автоматический переход в `Playing`.

## [x] 3. Реализация главного меню (`MainMenu`)

-   **Задача:** Создать визуальное представление для главного меню и экрана "Game Over".
-   **Результат:** Реализовано меню с использованием Bevy UI (Flexbox). Добавлены компоненты очистки.

## [x] 4. Реализация экрана "Game Over"

-   **Задача:** Улучшить экран завершения игры.
-   **Результат:** Реализован экран с отображением финального счета и инструкциями по перезапуску.

## [x] 5. Корректировка системы переходов между состояниями

-   **Задача:** Обеспечить правильную логику переходов между всеми состояниями игры.
-   **Результат:** Обновлена система переходов. Исправлена логика паузы. Добавлена очистка сущностей при переходе в `MainMenu`.
